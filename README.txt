Устанавливаем как любой репозиторий, clone($ git clone <URL> кланирует с гита проект), instal <directory>. Миграции все есть, так что выполняем $ php artisan migrate(и база готова), больше ничего делать не надо, файл .env я оставил в 
репозитории. Необходимо настроить точку входа через папку public. можете добавить .htaccess или переадресацыю на сервере.Не забудте прописать настройки Вашего MySql в файле ".env".По установки всё. 

Теперь по функционалу. 
Переход между страницами осуществляется кнопками на главной.
Для добавления книги или автора в каждом виде имеются кнопки с соответственным названием "Add".
Для редактированиея книги или автора вызыввается модальное окно нажатием на их название "link".
Удаление и добавление картинок проводится с помощью кнопок рядом с каждым автором или книгой.
